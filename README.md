# Gerenciador de tarefas simples 🖥️✨

Um monitor de sistema leve e direto para o terminal, feito em Python usando `rich` e `psutil`.

Eu queria uma ferramenta simples para rodar no terminal e acompanhar o uso da máquina (SO, CPU, Rede e arquivos locais) sem precisar abrir gerenciadores pesados ou interfaces complexas. O resultado foi esse painel modular e estilizado.

## 🛠️ O que ele faz?
- **Sistema:** Mostra o SO atual, o nome da máquina na rede (Node) e a quantidade de núcleos da CPU.
- **Rede:** Lista as interfaces ativas e seus respectivos endereços IP (IPv4).
- **Explorador de Arquivos:** Mostra os primeiros 10 arquivos do diretório atual para dar um feedback rápido de I/O.
- **Visual Organizado:** Divisão de tela feita com o sistema de layouts e painéis da biblioteca `rich`.
